MyNote - 极简桌面便笺 v1.0.0
================================

感谢您使用 MyNote！这是一个无边框、支持毛玻璃特效的极简桌面便笺工具。

【快速开始】
1. 直接双击运行 MyNote.exe 即可。
2. 首次运行会自动在同目录下生成 notes_data.json 用于存储数据。

【基本操作】
- 新增事项：在文本框输入内容后，按 Enter 键，或者点击底部的 "+" 号。
- 完成/取消：点击左侧方框，或按 Ctrl+Enter。
- 删除事项：右键菜单 -> 删除，或按 Ctrl+D。
- 调整位置：按住窗口任意空白处（或顶部区域）拖动。
- 调整大小：拖动窗口边缘（右侧或底部）即可缩放。
- 更多设置：在窗口上点击鼠标右键，可开启/关闭置顶、调整透明度等。

【快捷键列表】
- Enter       : 新建待办（换行请按 Shift+Enter）
- Ctrl + Enter: 切换完成状态
- Ctrl + D    : 删除当前项
- Ctrl + ↑/↓  : 上下移动当前项
- Ctrl + Z    : 撤销操作

【注意事项】
1. 数据文件：所有笔记保存在同目录下的 notes_data.json 文件中，请勿误删。
2. 杀毒误报：如果被杀毒软件拦截，请添加信任（因为软件未购买数字签名）。
3. 开机自启：如需开机启动，请为 exe 创建快捷方式，把它放入 Windows 的“启动”文件夹。
   (Win+R 输入 shell:startup 即可打开启动文件夹)
